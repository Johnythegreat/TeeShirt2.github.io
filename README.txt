TEE SHIRT — GITHUB PAGES VERSION

UPLOAD INSTRUCTIONS
1. Extract this ZIP.
2. Upload index.html, styles.css, app.js, and hero.png to the root of your GitHub repository.
3. In GitHub, open Settings > Pages.
4. Under Build and deployment, select "Deploy from a branch".
5. Select your main branch and the /(root) folder, then click Save.
6. Wait a few minutes and open the GitHub Pages link shown by GitHub.

This package uses plain HTML, CSS, and JavaScript. No npm install or build command is required.
