# Tee Shirt Full Working Website

Upload all files in this folder to Netlify, GitHub Pages, or any static hosting.

## Pages
- `index.html` = Customer page
- `admin.html` = Admin page

## Default Admin Login
Password: `admin123`

You can change the password inside Admin > Settings.

## Included Features
- Customer product catalog
- Search and filters
- Shopping cart
- Checkout with order number
- Order tracking
- Custom design request form with image upload
- Admin dashboard
- Product add/edit/delete
- Stock monitoring and quick stock update
- POS / walk-in sales
- Orders and status update
- Messages/custom requests
- Shop settings and shipping fee

## Important Note
This version uses browser localStorage so it works immediately after upload without Firebase or database setup. Data is saved in the browser/device you use. For multi-device real-time admin/customer syncing, connect it later to Firebase Firestore + Storage.
